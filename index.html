Geolocation

A small example of using Javascript and Google Maps API to work out the distance the user is from Belfast.

Farm Manager

This demo also utilizes Javascript and Google Maps to allow the user to draw over fields and record data to manage animal herds, crops, etc.

Image Detection

A simple demo that uses Open CV (Computer Vision) and Python to detect how many cattle are in the images.

Pi Bot

This small Robot utilizes a Raspberry Pi board with off-the-shelf components (includes ultrasonic sensor and motor-board) that once I bought and assembled, I then attached a webcam to the Robot to detect colour and change direction based on which side the colour fabric was on.

Online QR Scanner

A project (using open-source software) whereby I setup the ability to generate QR codes and using a conventional webcam to scan the barcodes, using Javascript and web browsers. This was an idea I have had for events like small festivals, were a user can run cheap entrance management services.

Snap Up!

A idea I had using Javascript and html5 features like video capture, to create a quick and easy product listing website, were the user can use their browser to snap a image of a product and add information as quickly as possible. The target audience would be Charity Shops with their need for single product listings.

Touchscreen Table

Using a old TV, Xbox Kinect and some off-the-shelf software, I recycled this old TV into a large touch device. Users would be able to use finger gestures like on touchscreen mobile devices to navigate the device with services like google maps and microsoft paint.

Tweet Map

Taking location data of tweets from Twitter's REST API, and overlaying the data over a Google Maps, I was able to create a simple search engine for tweets in a local area about any topic the User typed into the search.

Vidcord

Utilizing open-source software for html5 video capture, I created a simple Javascript video and audio recording device that could be used in Firefox browsers both on desktop and mobile, that allowed the user to record a chain of videos to be played back on the same screen.

VR Game Demo

Using Unreal Engine 4 and Oculus Rift Development Kit 2, I created a simple First Person Shooter Demo featuring enemy characters and destructible buildings.

Computer Visual

Using OpenCV, I was able to create a basic computer vision application that scans a photo and informs the user if they are walking in the right direction.

Scan & Go

A protoype I have been building in my spare time, inwhich a user can store digital receipts in the cloud using their unique QR Barcode that contains their account ID, for easy scanning and uploading on the go.

Augmented Reality Demo

Using AR.js and a webcam, a simple demo of Augmented reality using a browser to display Augmented 3D objects and images in the scene. Also using Wikitude (AR application) to display images at Geo-located points.

Machine Learning – Cow Classifier

By using Drone Imagery, OpenCV and Tensorflow, I created an application to count how many Cows exist in a single Image from the top-down perspective.

VR Home Visualizer

A quick demo I made using Virtual Reailty, Photogrammetry software and the Unity engine to visualize a building design. Please follow the link here: Google Slides
